# Websocket with AWS SQS 


## Getting Started

### ```Degine environments variables```
Write .env file
```
AWS_ACCESS_KEY_ID=
AWS_SECRET_ACCESS_KEY=
AWS_SESSION_TOKEN=
QUEUE_URL=
```

### ```Create python environment```
```
conda create -n poc-web-socket python=3.10
```

### ```Install dependencies```
```
pip -r install requirements.txt
```
